# كيفية الاتصال بقاعدة البيانات وفحصها

## الخطوات:

### 1. الاتصال بقاعدة البيانات
في PowerShell، نفذ الأمر التالي:
```bash
flyctl postgres connect -a emingo-db
```

### 2. بعد الاتصال
ستظهر لك نافذة psql مثل:
```
postgres=#
```

### 3. نسخ الاستعلامات
- افتح ملف `quick-check.sql` أو `inspect-db.sql`
- حدد الاستعلام الذي تريده
- انسخه (Ctrl+C)

### 4. لصق الاستعلام في psql
- انقر بزر الماوس الأيمن في نافذة psql
- أو اضغط Shift+Insert
- أو Ctrl+Shift+V (في بعض المحطات)

### 5. تنفيذ الاستعلام
- اضغط Enter بعد لصق الاستعلام
- ستظهر النتائج

### 6. الخروج من psql
اكتب:
```
\q
```
ثم اضغط Enter

## نصائح:
- يمكنك نسخ استعلام واحد في كل مرة
- أو نسخ عدة استعلامات معاً (يفصل بينها `;`)
- استخدم `\q` للخروج من psql
